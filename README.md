# **A React-based blog in development**

## 🖋️ 项目简介

一个基于react的响应式个人blog搭建...

## 🛠️ 功能版本

- *`node.js`: 18+*
- `react`: 18.3.1
- `antd`: 5.18.0
- `axios`: 1.7.2
- `react-router-dom`: 6.23.1
- `redux`: 5.0.1
- `sass`: 1.77.4
- `highlight.js`: 11.9.0
- `jinrishici`: 1.0.6
- `marked`: 13.0.0
- `gray-matter`: 4.0.3
- `echarts`: 5.5.0
- ...

## 🔑 使用
### 新建文章
- `npm run create-post "文件名"`
### 构建/运行前
- `npm run gen-posts-config`
