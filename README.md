# **A React-based blog in development**

## 🖋️ 项目简介

一个基于react的响应式个人blog搭建...

## 🛠️ 功能版本

- *`node.js`: 18+*
- `react`: 18.3.1
- `antd`: 5.18.0
- `axios`: 1.7.2
- `react-router-dom`: 6.23.1
- `sass`: 1.77.4
- `redux`: "^5.0.1"
- `redux-saga`: "^1.0.3"
- `highlight.js`: 11.9.0
- `marked`: 13.0.0
- `gray-matter`: 4.0.3
- `echarts`: 5.5.0
- ...

## 🔑 使用
### 安装依赖
- `npm install`
### 新建文章
- `npm run create-post "文件名"`
### 生成文章配置文件
- `npm run gen-posts-config`
### 启动应用
- `npm start`
### 构建应用
- `npm build`
